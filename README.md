## Getting Started
These instructions will get you a flying drone.

### Prerequisites
2 arduinos 

4 motors 

4 motorcontrollers 

2 wireless receivers/transmitters

### Installing
Download the source code.
```
git clone https://github.com/AbdelhalemKassara/Self_Leveling_Quadcopter.git
```
